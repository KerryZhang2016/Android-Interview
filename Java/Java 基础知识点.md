###Java 基础知识点
可以用几段话说明清楚的Java基础知识点，没有特别的分类，以下进行逐条描述。   
  
- 基本数据类型

| 基本数据类型    | 大小           | 对应封装类型   |
| --------------|:-------------:|:------------:|
| int           | 4字节          | Integer      |
| double        | 8字节          | Double       |
| long          | 8字节          | Long         |
| float         | 4字节          | Float        |
| short         | 2字节          | Short        |
| byte          | 1字节          | Byte         |
| character     | 2字节          | Character    |
| boolean       | 1比特          | Boolean      |
